## Установка и запуск

1. Клонировать репозиторий:

```
git clone https://github.com/darkfated/darkfatedru.git
cd darkfatedru
```

2. Установить зависимости:

```
npm install
```

3. Запустить сервер разработки:

```
npm run serve
```

4. Для сборки проекта:

```
npm run build
```
