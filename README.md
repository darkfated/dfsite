## Установка и запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/darkfated/dfsite.git
cd dfsite
```

2. Установить зависимости:

```
npm install
```

3. Запустить сервер разработки:

```
npm run serve
```

4. Для сборки проекта:

```
npm run build
```
